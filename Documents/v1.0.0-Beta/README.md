# v1.0.0-Beta

## ⭐ 新增新能

- 支持 `数据表格` 无级框选，类似 `Excel` 矩形框选效果。
- 支持导航区工作线程与 UI 线程隔离，并显示加载中。
- 支持SQL编辑器终止当前正在运行的查询任务。
- 新增SQL脚本执行器`日志记录`打印功能。
- 支持SQL脚本`全局美化`功能。
- 支持复制数据表格中的内容。
- 支持在表格列上显示 `字段类型` 以及 `是否主键`。
- 新增表格数据刷新功能，支持任意表格刷新。
- 新增执行器日志面板清空内容按钮。
- 新增设计表结构和重建索引功能。
- 新增表字段结构修改功能，支持函数、字符等默认值。
- 支持表结构设计主键修改，以及支持按顺序添加主键。
- 支持`查询脚本`重命名。
- 支持复制、打开`查询脚本`所在路径。
- 新增表结构字段删除功能。
- 支持`表索引`新增、修改、删除功能，以及设置索引可见性。

## 👻 优化功能

- 优化 `Dialog` 之间的按钮边距，避免排版过于紧凑。
- 禁止表节点选中后跳转详情页。
- 合并数据库 `打开` 和 `关闭` 右键菜单按钮。
- 优化表格刷新逻辑，新增表格刷新时 `背景闪烁`。
- 代码行数超出 `5000` 行后不做高亮渲染。
- 优化大量数据删除时界面卡顿问题。
- 预览页面不可关闭结果集标签。
- 优化预览页面工具按钮位置，与其他页面对齐。
- 优化设计表编辑保存报错后不清空已编辑的内容。
- 优化提示框，支持复制错误提示框中的错误信息。
- 支持`字段元数据`信息`防篡改`校验，避免提交多余无意义语句。
- 支持脚本自动保存。
- 优化`代码高亮`逻辑，优化`SQL编辑器`性能。

## 🐞 BUG 修复

- 修复查询时错误提示框重复弹出问题。
- 修复打开 SQL 脚本时连接和数据库选择错误问题。
- 修复 `关闭其他标签` 时会调用当前聚焦标签关闭事件问题。
- 修复刷新预览表格出现幽灵行问题。
- 修复`执行器日志`标签关闭不可打开问题。
- 修复中文排序问题，支持拼音 A-Z 顺序
- 修复在无主键的表中删除数据会导致多条相同数据被删除。
- 修复设计表结构中 `specs` 为 `null` 时导致设计表页面打不开问题。
- 修复设计表结构中 `default` 为 `null` 时，导致某些类型新增失败问题。
- 修复 MySQL 中 `BIT` 类型显示不正确问题。
- 修复带参数类型的函数默认值无法正确解析问题。
- 修复查询页预览框标签全部关闭后容器没关闭问题。
- 修复当单元格编辑器丢失焦点后没提交编辑内容。
- 修复表结构设计页面重复打开问题。
- 修复当编辑`连接名称`后，在`新建查询`面板选择连接时会出现旧名称连接对象。
- 修复表结构设计默认值为空值时导致`SQL`语句构建不正确。
- 修复对话框打开位置与主窗口所在位置不一致问题。
